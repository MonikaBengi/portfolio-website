## How Itâ€™s Built
- **Navbar:** fixed at the top, animated fade-in with keyframes  
- **Hero video:** full-width background, hidden on small screens for performance  
- **Slideshow:** `.mySlides.fade` with arrows and dots; `showSlides()`, `plusSlides()`, `currentSlide()` and auto-rotate with hover pause and tab visibility handling  
- **Contact form:** modal toggled by `openForm()` / `closeForm()`, closes on outside click  
- **Typography:** viewport-based scaling and multiple breakpoints
